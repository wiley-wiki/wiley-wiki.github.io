# 字符串

# 数据类型

类型 长度 备注
char 0 ~ 255字节 指定长度字符串
varchar 0 ~ 65535字节 动态可变长度字符串
tinyblob 0 ~ 255字节 不超过255个字符的二进制字符串
tinytext 0 ~ 255字节 短文本字符串
blob 0 ~ 65535字节 二进制形式的长文本数据
text 0 ~ 65535字节 长文本数据
mediumblob 0 ~ 16777215字节 二进制形式的中等长度文本数据
mediumtext 0 ~ 16777215字节 中等长度文本数据
longblob 0 ~ 4294967295字节 二进制形式的极大文本数据
longtext 0 ~ 4294967295字节 极大文本数据

字符串分为二进制和非二进制类型

二进制用于存储图片、声音等二进制文件

非二进制文本用于存储文本数据

非二进制文本受字符集和校对规则影响

# 字符集

# 查看服务器支持的字符集

show character set;
1

数据表不设置字符集则继承数据库的字符集，字段不设置字符集则继承数据表的字符集

# 校对规则

在字符集内用于字符比较和排序的一套规则

以_ci结束的为不区分大小写

以_bin结束的为区分大小写

校对规则会影响到排序和比对

# 查看系统支持的校对规则

show collation;
1

修改表校对规则，对表的原有字段并不影响，只对新增的字段影响。

# 数值

# 整型

类型 长度
tinyint 占1个字节(-128 ~ 127)
smallint 占2个字节(-32768 ~ 32767)
mediumint 占3个字节(-8388608 ~ 8388607)
int 占4个字节(-2147483648 ~ 2147484647)
bigint 占8个字节(+-9.22*10的18次方)

若设置字段unsigned，则该数值类型最大值翻倍，如tinyint字段设置了unsigned属性，则取值范围为(0 ~ 256)

字段设置长度，不是定义该字段为可输入几个字符而是显示长度

设置 zerofill 自动填充0

添加有前导零字段和不为负数字段

alter table student add status tinyint(2) zerofill unsigned not null;
1

# 浮点型

类型 长度 范围(有符号) 范围(无符号)
float 占4个字节 (-3.402 823 466 E+38，-1.175 494 351 E-38)，0，(1.175 494 351 E-38，3.402 823 466 351 E+38) 0，(1.175 494 351 E-38，3.402 823 466 E+38)
double 占8个字节 (-1.797 693 134 862 315 7 E+308，-2.225 073 858 507 201 4 E-308)，0，(2.225 073 858 507 201 4 E-308，1.797 693 134 862 315 7 E+308) 0，(2.225 073 858 507 201 4 E-308，1.797 693 134 862 315 7 E+308)
decimal DECIMAL(M,D) ，m<65 是总个数，d<30 依赖于M和D的值 依赖于M和D的值

float类型 2^23 = 8388608，一共7位数，但是能绝对保证精度的只有6位

double类型 2^52 = 4503599627370496，一共16位数，double的精度位数为15位

浮点型(float double)在数据库中存储的是近似值，而定点类型1在数据库中存储的是精确值

定点数 decimal(10,2)参数 m < 65，d < 30 且 d < m，d是小数位

对货币或者精度要求很高的数据，应该采用定点数decimal数据类型存储

# ENUM/SET

# ENUM

ENUM类型只允许在预定义的集合中取一个值，枚举最大可以有65535个

alter table students modify sex enum("男","女") not null;
1

索引 1 是男，索引 2 是女

添加数据

insert into students(sname,sex) values('李岗',1,'男');
insert into students(sname,sex) values('李玉',1,2);
1
2

查询数据

select * from students where sex = '女';
select * from students where sex = 2;
1
2

# SET

SET类型可以从预定义的集合中取任意数量的值，最多可以有64个

# 使用场景

增加数据

alter table article add flag set("置顶","图文","推荐","热门");
1
insert into article(title,status,flag) VALUES('这是一个文章标题',1,'图文,推荐,置顶');
1

查询数据

select * from article where find_in_set("置顶",flag);
1
