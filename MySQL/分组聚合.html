# 分组聚合

# 什么是分组聚合

举个例子：将某个学校的班级进行分组，按班级进行分组，然后将所有班级分组情况统计就叫聚合

select * from students group by class;
1
ERROR 1055 (42000): Expression #1 of SELECT list is not in GROUP BY clause and contains nonaggregated column 'teacher.students.id' which is not functionally dependent on columns in GROUP BY clause; this is incompatible with sql_mode=only_full_group_by
1

执行上面语句发现报错，是因为受 sql_mode 模式 only_full_group_by 影响，group by 中出现的字段必须也要在字段列表中出现

mysql> select class,count(*) from students group by class;
+-------+----------+
| class | count(*) |
+-------+----------+
| 1     |        3 |
| 2     |        3 |
| 3     |        2 |
+-------+----------+
3 rows in set (0.00 sec)
1
2
3
4

6
7
8
9

使用 count 函数即可解决问题

或者将 sql_mode 模式进行修改

set sql_mode = (select replace(@@sql_mode,'ONLY_FULL_GROUP_BY',''));
1

# 多字段分组

统计每个班级的学生年龄在班级有几人

select count(*),class,age from students group by class,age;
1

# HAVING

HAVING语句通常与GROUP BY语句联合使用，用来过滤由GROUP BY语句返回的记录集。 HAVING语句的存在弥补了WHERE关键字不能与聚合函数联合使用的不足。

使用分组查询班级人数大于2人的班级

select class,count(*) from students group by class having count(*) > 2;
1

上面语句也可以起别名

select class,count(*) as c from students group by class having c > 2;
1
